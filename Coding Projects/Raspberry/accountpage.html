Game Design

The Basics of Blockly

Learn the basics of coding Blockly and how it relates to other languages such as JavaScript with Google Developer's coding interface and in-depth tutorials and other activities Scratch and CODE.

Advanced Mechanics

Learn the more advanced mechanics of Scratch, such as using Cloud Data to enhance your projects and creating custom art and music to go along with your game.

Planning out your Game

Turn your idea for a game into a reality. Learn how to create an "outline" for your game and how to create special mechanics that make your game unique and stand out.

Project: Create your Game

Show off your skills and finally create your game and share it on Scratch. Learn how to finalize your game and how to share it with a community.

Beginner Web Design

HTML5

Learn the language that is behind almost every website. Know the graphical and structural elements of HTML tags and what each one does or can do when supplemented with another language.

CSS3

Learn the how to beautify your HTML using the latest version of CSS. Also learn how to support create browser-specific styles using -webkit- and -moz-, implement custom fonts, and use external style sheets such as Font Awesome and Bootstrap.

JavaScript

Implement your blockly skills into your website. Create new mechanics and give even more functionality to your websites using JavaScript. Learn how to use DOM, functions, cookies, variables, and more.

Project: DIY Website

Finally create your own website. Make anything from a blog to a website for a new online product you've developed. Be sure to use HTML, CSS, and JavaScript in your website.

Digital Office Skills

Know your OS

Learn the ins and outs of Windows 10 as well as small tricks to help your improve efficiency. Also we'll teach you some basic information about the command prompt, how Windows handles unknown files, and how program (.vbs and .exe) files work.

Online 365

Learn what each of the Office 2016 and Office Online applications do, what they're for, and how to use them. Learn how to use the cloud (OneDrive) to your advantage for everything from computer backup to quickly sharing files with friends.

Design Tips

Learn how to create a cool design for your Excel tables, PowerPoint presentations, Word documents, and more. Also learn how to create and save templates, even with custom fonts and color schemes included.

Project: Get Hired

Use Word to create a resume for a future job that you want to have. Include goals that you want to complete in the future that may help you obtain the job, as well as achievements that an employer may be looking for that will set you apart from others.
